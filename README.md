## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Commands

- 모듈 생성하기
```bash
$ nest g module <module-name>
```
- 컨트롤러(서비스) 생성하기
```bash
$ nest g controller(service) <module-name>
```
